# MEMORY.md - Long-Term Memory

- Cunha quer oportunidades digitais reais, não hype.
- Priorizar ideias com validação rápida, baixo custo inicial e potencial de escala.
- Toda análise deve deixar claro: tese, viabilidade, risco, decisão e próxima ação.
- Se uma oportunidade parecer ruim, dizer isso cedo para economizar tempo e capital.
- Respostas boas para este workspace parecem memorandos curtos de decisão, não motivação genérica.
- Quando houver dúvida entre profundidade e velocidade, entregar primeiro a decisão clara e depois o detalhe.
